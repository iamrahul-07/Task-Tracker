# Task Tracker App (MERN)

A full-stack Task Management application built using MongoDB, Express.js, React, and Node.js.

## Features
- Create tasks
- Update status
- Delete tasks
- Priority levels (Low/Medium/High)
- Due date tracking
- Fully responsive UI
- Tailwind CSS styling

## Tech Stack
Frontend: React + Tailwind  
Backend: Express + Node  
Database: MongoDB Atlas  

## API Endpoints
POST /api/tasks  
GET /api/tasks  
PUT /api/tasks/:id  
DELETE /api/tasks/:id  

## How to Run

Backend:
npm install
npm start

Frontend:
npm install
npm run dev
